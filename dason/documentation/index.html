Introduction

Thank you for purchasing Dason Admin & Dashboard Template.

Dason is a simple and beautiful admin dashboard template built on top of awesome Python 3.12.0,Django 4.2.6 & Bootstrap v5.3.2. It's crafted using modern web technologies HTML5, CSS3, Gulp , Yarn and jQuery.

Dason is an admin dashboard template that is a beautifully crafted, clean & minimal designed admin template with Dark, Light Layouts with RTL options. You can build any type of web application like Saas based interface, eCommerce, CRM, CMS, Project management apps, Admin Panels, etc of any project.

Dason Admin & Dashboard Template is easily customizable and developer-friendly codes. It will help your team moving faster and saving development costs and valuable time.
